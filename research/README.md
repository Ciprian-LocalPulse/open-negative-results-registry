# Research

Evaluations, audits, and reflective/foundational essays about the project —
kept separate from the operating documentation in [`docs/`](../docs/) and the
technical specifications in [`architecture/`](../architecture/) and
[`specifications/`](../specifications/), since these documents assess or
reflect on the project rather than define how it runs.

## Academic Reviews

- [`academic-reviews/onrr_academic_maturity_audit.html`](academic-reviews/onrr_academic_maturity_audit.html) —
  an academic audit and maturity matrix for the registry.

## Essays

- [`essays/Founders_Statement.md`](essays/Founders_Statement.md) — the
  founder's statement on why this project exists.
- [`essays/The_Epistemology_of_Silence.md`](essays/The_Epistemology_of_Silence.md) —
  a foundational essay on symmetrical evidence and the registry's
  epistemic rationale (renamed from `The Epistemology of Silence.md` to
  remove spaces; content unchanged). Previously existed as two identical
  copies (repository root and `docs/`); this is the single copy kept.
- [`essays/the_epistemology_of_silence_spa.html`](essays/the_epistemology_of_silence_spa.html) —
  a standalone interactive rendering of the essay above (moved here from
  `site/`, where it was not linked from the search interface).
