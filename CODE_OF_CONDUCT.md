# Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in this
project a harassment-free experience for everyone, regardless of career
stage, institution, nationality, gender, background, or level of technical
experience. This project depends on contributions from clinicians and
researchers who may have never used GitHub before — patience and clarity are
part of the mission, not an afterthought.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful of differing viewpoints, including disagreements about
  methodology or classification of a negative result
- Giving constructive feedback on data submissions
- Focusing on what is best for the reliability of the shared dataset
- Showing patience toward non-technical contributors

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks in Issues, Pull Requests, or
  any project space
- Submitting deliberately false, fabricated, or misleading data
- Publishing others' private information (including patient data) without
  consent
- Any conduct which could reasonably be considered inappropriate in a
  professional or academic setting

## Data Integrity Is Part of This Code

Because this project's value depends entirely on trustworthy data, knowingly
submitting fabricated negative results, plagiarized content, or
misattributed findings is treated as a severe violation, in addition to any
academic misconduct policies at a contributor's own institution.

## Enforcement

Maintainers are responsible for clarifying standards and may remove, edit,
or reject contributions, comments, and other interactions not aligned with
this Code of Conduct. Repeated or severe violations may result in a
temporary or permanent ban from the project's spaces.

## Reporting

Report unacceptable behavior by opening a confidential Issue marked
`conduct` or by contacting the maintainer directly through the contact
method listed in the repository README.

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.
